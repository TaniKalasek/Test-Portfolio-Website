Dovednosti

Základní webové technologie

HTML

HyperText Markup Language

HTML je základní kostra každé webové stránky, která definuje její strukturu a obsah.

Používá tagy (značky) k označení prvků, jako jsou nadpisy, odstavce, obrázky a odkazy. Slouží k sémantickému uspořádání informací a určuje, co je na stránce zobrazeno.

CSS

Cascading Style Sheets

CSS je jazyk, který se stará o vzhled a vizuální prezentaci obsahu definovaného v HTML. Řídí styly, jako jsou barvy, písma, rozvržení (layout) a animace prvků.

Díky CSS je možné oddělit strukturu od designu a zajistit tak konzistentní a atraktivní vzhled.

JavaScript

JS

JavaScript dodává webovým stránkám interaktivitu a dynamické chování. Umožňuje reagovat na akce uživatele (např. kliknutí), měnit obsah na stránce v reálném čase a komunikovat se serverem. Je to programovací jazyk, který "oživuje" statický obsah.

Nástroje a Workflow

Git

Verzování kódu

Git je systém pro sledování změn v kódu a pro spolupráci v týmu. Umožňuje bezpečně ukládat různé verze projektu (tzv. "commity"), vracet se k předchozím stavům a plynule integrovat práci více vývojářů dohromady.

Zajišťuje, že veškerá historie projektu je zachována a v případě chyby lze kód snadno obnovit.

Terminál

Příkazová řádka

Terminál (neboli Příkazová řádka) je textové rozhraní, které umožňuje rychle a efektivně spravovat soubory, spouštět programy a provádět operace na serveru nebo v projektu.

Pro developery je nepostradatelný pro práci s Gitem, instalaci závislostí (např. přes npm nebo Composer) a pro spouštění lokálního vývojového serveru.

Osvedčené postupy

Debugging, testování

Debugging(ladění) je proces systematického vyhledávání a opravování chyb v kódu, aby aplikace fungovala správně.

Testování zase představuje ověřování funkčnosti aplikace – od manuálních zkoušek až po automatizované testy – čímž je zaručeno, že software splňuje požadavky a je spolehlivý.
